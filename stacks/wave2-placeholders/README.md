# Wave 2 Placeholders

## Status

Active as a temporary landing surface only. This stack provides distinct placeholder pages for deferred Wave 2 services.

## Intended Role

- give Homepage cards a real per-service destination
- avoid routing undeployed services back to the shared portal
- provide a clear coming-soon message until each real service is deployed
